Sobre nosso projeto

Os dados mundiais sobre o COVID-19 são de extrema importância porque fornecem informações cruciais sobre a propagação da doença em todo o mundo. Esses dados incluem o número de casos confirmados, mortes, recuperações, hospitalizações e outros fatores relevantes que ajudam a entender a dimensão da pandemia.

Nosso Objetivo

Temos como objetivo conscientizar a população brasileira sobre os demais dados apresentados, dando a margem de casos confirmados, órbitos e recuperados, para assim manter a todos telespectadores informados dos acontecimentos atuais.

Ferramentas ultilizadas para formação do projeto

As principais ferramentas ultilizadas para a formação do nosso projeto e nosso desenvolvimento foram os apps: Visual Studio Code - Responsável pela criação do site; Google Charts - Responsável pelos gráficos aplicados na página inicial.
